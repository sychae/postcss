# 설치

```
git    : https://git-scm.com/download
nodejs : https://nodejs.org/ko/ 안정적 버전
yarn   : https://yarnpkg.com/lang/en/docs/install/
vscode : https://code.visualstudio.com/
```

# CLONE

```
git clone https://github.com/sychae/postcss.git
cd postcss

```

# 설정 / 프로젝트 시작.

```
rd -r .git
git init

yarn install

config.js
```


# BUILD 및 HOT 실행

```
yarn run watch
yarn run serve
yarn run build

```
